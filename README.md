# gotools
go tools

# gotestfile
Create template of test.go file

## Installation
```
$ go get github.com/hiromaily/gotools/gotestfile
```

## Usage
```
Usage: gotestfile [options...]

Options:
  -n     package name

e.g.
 $ gotestfile -n new-package-name
```
